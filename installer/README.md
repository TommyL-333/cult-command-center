# TikTok Shop Content Pipeline — Claude Skill

A Claude Code skill that builds a complete TikTok Shop content pipeline for any brand.

## What it builds

A self-hosted dashboard that runs the full content workflow:

**Brief → Trend Analysis → Script Ideas → AI Scripts + Voice Feedback → Video Upload → Publish**

- 🛍 **TikTok Shop publishing** via [Storista](https://storista.io/?ref=tommy)
- 📱 **Multi-platform scheduling** via [Buffer](https://join.buffer.com/tommy-lynch)
- 🎭 **AI video generation** via [Arcads](https://arcads.ai/?via=tommy-lynch)
- 📊 **TikTok analytics** via Apify
- 🔒 **Team access** via Cloudflare Access (free Google OAuth)
- 🚂 **Deploys to Railway** with persistent storage

---

## Install

1. Download **`SKILL.md`**
2. Create this folder on your computer:
   ```
   ~/.claude/skills/tiktok-shop-content-pipeline/
   ```
3. Drop `SKILL.md` into it

That's it.

---

## Use

Open Claude Code in a new empty folder and say:

> "Build me a TikTok Shop content pipeline"

Claude will guide you through the full setup — account signups, brand configuration, building the app, and deploying it.
